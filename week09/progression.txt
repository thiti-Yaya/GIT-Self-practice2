Event
    - สถานะที่เกิดขึ้นบนหน้าapplication หรือสถานะของWebเปลี่ยน
        - เช่น ผุ้ใช้คลิกเมาส์,ผู้ใช้มาพิมพืแป้นพิมพ์(Userมีปฎิสัมพันธ์), หน้าWebโหลด(Userมีปฎิสัมพันธ์)

Event Propagation ทิศทางการไหลของเหตุการณ์
    - Three Phases Event Flow
        - In the capturing Phase 
            - บนลงล่าง 1.document 2.html 3.body 4.div
        - In the target Phase
            - จุดที่คลิก ตัวelement
        - In the Bubbling Phase
            - ล่างขึ้นบน 1.div 2.body 3.html 4.Document

Way of using Web Event
    - Event Handlers Properties (ไม่ให้ใช้)
        - วิธีเก่า เป็นการเอาjsไปรวมกับhtml
    - Adding and removing event Handlers
        - ถ้าจะเพิ่มevent ให้ใช้งานผ่านตัว addEventListener()
        - ถ้าจะลบevent ให้ใช้งานผ่านตัว removeEventListener()
        - 3 parametorที่สำคัญเวลาเราจะใช้
            - 1. event type ให้ใส่เป็นparametorตัวเเรก
            - 2. ให้ระบุเมื่อเกิดeventนั้นเราจะให้มันทำอะไร ก็คือให้เขียนfunction เป็นparametorตัวที่2
            - 3. ให้ ระบุว่าeventมันจะทำงานแบบ บนลงล่าง หรือ ล่างขึ้นบน default คือ bubble phase(false) ถ้าจะเปลี่ยนเป็นcapture phaseให้ใช้(true)
        - ตัวอย่างการเขียน
            const btn = document.getElementById("myBtn")
            btn.addEventListener("click", () => {
                console.log(btn.id)
            }, false)

    - Adding multiple event handlers (addEventListener())
        - add event 1objectเดียวกันสามารถadd event ชื่อevent type เดียวกันได้มากกว่า1
            - เช่น 
                const btn = document.getElementById("myBtn")
                btn.addEventListener("click", () => {
                console.log(btn.id)
            }, false)
               btn.addEventListener("click", () => {
                console.log("Hello World!")
            }, false)
    - Removing event handlers (removeEventListener())
        - ข้อจำกัดของremoveEvent 
            - เวลาจะremoveEventให้เขียนparametorแบบเดียวกับที่ add eventมา และใช้ไม่ได้กับanonymous function
            - เช่น
                const btn = document.getElementById("myBtn")
                let handler = function() {
                    console.log(btn.id)
                }
                btn.addEventListener("click", handler, false)
                btn.removeEventListener("click", handler, false)
Event Properties
    - eventPhase
        - มันจะคืนค่าเป็นเลข 1 2 3 ตาม Flow ที่มันไหล
            - 1 หมายถึงมันจะอยู้ที่ capturing phase
            - 2 หมายถึงมันจะอยู้ที่ target phase
            - 3 หมายถึงมันจะอยู้ที่ Bubbling phase
    - target
            - ตัวที่ถูกคลิกจริงๆ (element ที่เกิดเหตุการณ์จริง)
    - currentTarget
            - “ตัวที่ event listener ถูกผูกอยู่”
    - preventDefault()
        - หยุดพฤติกรรมปกติของ element”
            - เช่น ไม่ให้ลิงก์เปลี่ยนหน้า, ไม่ให้ฟอร์มส่งข้อมูล, ไม่ให้ติ๊ก checkbox

Event type 
    - Event Categories
        - State change events
            - ใช้ตรวจจับ “การเปลี่ยนแปลงสถานะของหน้าเว็บหรือ element”
            - ตรวจการเปลี่ยนแปลงของหน้า/องค์ประกอบ
        - Focus events  
            - เกิดเมื่อ element ได้รับหรือเสีย “focus” 
                - เช่น cursor อยู่ในช่อง input
        - Mouse events
            - ใช้ตรวจจับการคลิก การเลื่อน หรือการเคลื่อนเมาส์
            - การคลิกหรือเคลื่อนเมาส์
        - Keyboard events
            - ใช้ตรวจจับการกด/ปล่อยปุ่มคีย์บอร์ด
            - การกดหรือปล่อยปุ่ม
        - Input events
            - ตรวจจับเมื่อค่าของช่อง input, textarea, หรือ select เปลี่ยนระหว่างพิมพ์
            - การพิมพ์หรือส่งฟอร์ม
